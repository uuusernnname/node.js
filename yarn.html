ESLint

  1. ESLint хранит настройки в файле .eslintrc.json

  2. Также нужен файл .eslintignore

  3. Установка с помощью yarn add --dev eslint

  4. Формируем набор праил yarn run eslint --init

  5. Создадим отдельный скрипт в package.json "lint": "eslint ."

  6. Запускаем линтер yarn run lint

  7. Для того, чтобы исправить проблемы yarn run lint -- fix

Babel

  1. Устанавливаем babel yarn add --dev @babel/core @babel/cli @babel/preset-env

  2. Конфигурационный файл .babelrc

  3. Добавляем в package.json "build": "babel src -d dist"

  4. Запускаем транспайлинг yarn run build

  5. Файл с именем .browserslistrc — позволяет установить, поддержку
    каких браузеров (окружений) необходимо обеспечивать,

JEST

  1. Установка с помощью yarn add --dev jest babel-jest @babel/core @babel/cli @babel/preset-env @babel/polyfill

  2. Пропишем скрипт в package.json "test": "jest --passWithNoTests"

  3. Добавить в "env": .eslintrc.json "jest": true

  4. Запуск автотестов yarn run test

Webpack

  1. Установка yarn add --dev webpack webpack-cli babel-loader path

  2. В скрипты package.jcon "build": "webpack --mode production"

  3. Запуск yarn run build

  4. Плагин для сборки HTML yarn add --dev html-webpack-plugin html-loader

  5. Плагин для сборки CSS yarn add --dev mini-css-extract-plugin css-loader

  6. Плагин для сборки изображений yarn add --dev url-loader file-loader

  7. Установка Webpack Dev-server yarn add --dev webpack-dev-server

  8. В скрипты "start": "webpack-dev-server --open --mode development"

  9. Запуск yarn start

  10. Скачать конфигуратор webpack

JSDoc

  1. Установка yarn add --dev jsdoc

  2. В скрипты package.jcon "doc": "jsdoc -c jsdoc.conf.json"

  3. Запуск yarn run doc

Corejs

  1. Установка yarn add --dev core-js

  2. Изменить параметры в .babelrc см. ниже

Создайте конфигурационный файл JSDoc - jsdoc.conf.json

{
 "source": {
  "include": ["src/js"]
 },
 "opts": {
  "encoding": "utf8",
  "destination": "docs",
  "recurse": true
 }
}

Если jest выдает ошибку definde-property, надо поменять в .babelrc

{
 "presets": [["@babel/preset-env", {
  "useBuiltIns": "entry",
  "corejs": 3
 }]]
}
